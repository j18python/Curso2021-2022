Análisis realizado

• Origen de los datos → Ayuntamiento de Madrid (Link al portal de datos abiertos del ayuntamiento de Madrid)

• Nombre del fichero .csv a emplear → parquimetros.csv

• Tabla con las columnas del CSV:

Columna Tipo de variable Comentario
Matricula int Identificador del parquímetro
GIS_X float Coordenada del eje X
GIS_Y float Coordenada del eje Y
Fecha_alta string Fecha en la que se dio de alta el parquímetro
Distrito string Distrito de Madrid al que permetece el parquímetro
Barrio string Barrio (dentro de un distrito de Madrid) al que pertenece el parquímetro
Via string Tipo de via de Madrid en la que se encuetra el parquímetro
Num_finca string Numero (dentro de la calles de Madrid) donde se encuentra el parquimetro

Dataset Requirements

R1. Los datos provienen de un dataset de la ciudad de Madrid.

R2. El dataset se encuentra en formato CSV

R3. Las condiciones generales de la licencia permiten la reutilización de la información para fines comerciales y no comerciales.

R4. Los datos pueden ser facilmente relacionados con zonas geográficas de la ciudad de Madrid.
