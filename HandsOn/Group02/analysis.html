Data Analysis:

Accident String It is a string data type composed by letters and numbers to identify the accident.
Crossing String It provides information about crossing, where there are crossings with traffic lights to avoid accidents.
District String It provides information about the zone of the accident.
Address String It contains the street and number where the accident was produced.
Geometry String It is a string which provide the latitude and the longitude for the crossings and traffic lights for bicycles.
TrafficLight String It provides information about where are the traffic lights and when were installed.
Person String It indicates the type of person who was affected by the accident.
