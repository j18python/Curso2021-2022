REQUISITOS

Primer requisito

Los conjuntos de datos usados proceden de datos.madrid.es. Por tanto, dado que este portal forma parte del proyecto de Ciudades Abiertas, consideramos que cumplimos esta condición.

Segundo requisito

Los conjuntos de datos escogidos (datos en tiempo real sobre la calidad del aire y datos sobre las estaciones de control) están disponibles bajo el formato CSV. Por tanto, consideramos que cumplimos el segundo requisito.

Tercer requisito

La Licencia de los datos de datos.madrid.es están disponibles en sus Condiciones de Uso. En ellas, se establece que se permite la reutilización de los documentos para fines comerciales y no comerciales, por lo que este tercer requisito también se vería cumplido.

Cuarto requisito

El conjunto de datos sobre las estaciones de control (informacion_estaciones_red_calidad_aire.csv) posee los nombres de los lugares en los que se encuentra, junto con la información de sus coordenadas, y un número identificador en la columna CODIGO_CORTO, el cual está relacionado con la columna ESTACION del conjunto de datos en tiempo real (horario.csv), y podrá ser usado para relacionarse con otros conjuntos de datos que posean estos identificadores. Además, al poseer datos de localización, se podría relacionar con entidades de otros conjuntos que compartan localización o que esta sea cercana. Por tanto, los datos de ambos conjuntos pueden relacionarse fácilmente entre ellos y con entidades del mundo real, cumpliendo así este requerimiento.

Quinto requisito

Existe un documento pdf que adjuntamos aquí el cual es una guía para realizar una buena interpretación sobre los datos de la calidad del aire. Por tanto, consideramos que este quinto requisito se cumple en buena medida

Sexto requisito

Para este proyecto, hemos decidido usar dos conjuntos de datos estrechamente relacionados entre sí, que se complementan y proceden del mismo portal. Por tanto, ya que se tratan dos fuentes de datos distintas, sí que cumplimos este requisito opcional; no obstante, dada la relación entre ambos conjuntos, esta condición no se cumple en la misma medida que las demás.
