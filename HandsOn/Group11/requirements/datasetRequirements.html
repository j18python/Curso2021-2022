Dataset Requirements


R1: Data are in one of the selected smart city domains

Every datasets have been selected from Madrid's city hall open data web. You can find the datasets at the end of the page.


R2: Data are available as a CSV file

Every datasets are available as a CSV file.


R3: Data have an open license so they can be published

Every datasets have the same open license, which states that it is allowed to reuse the documents for commercial and non-commercial purposes. You can check the license here.


R4: Data can be easily linked with generic real-world entities

Every field of the datasets can be linked with real-world entities like Madrid's neighborhoods, streets, areas, districts, etc...


R5: Documentation exists for the data

Every datasets contain more information, which is available in the links below.
