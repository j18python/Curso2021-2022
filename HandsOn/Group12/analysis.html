Análisis de los datos seleccionados

En los ficheros csv podemos ver los diferentes valores de datos

  1. "PROVINCIA":

    Son las provincias de Galicia (A Coruña, Pontevedra, Lugo)

    2. Tipo: String

  2. "CÓDIGO-PROVINCIA"

    Especifica una de las provincias mencionadas en Galicia

    3. Tipo: Integer

  3. "AYUNTAMIENTO"

    Nombra el municipio de la provincia mencionada

    4. Tipo: String

  4. "CÓDIGO-AYUNTAMIENTO"

    Código del municipio

    5. Tipo: Integer

  5. "PLAYA"

    Nombre de la playa

    6. Tipo: String

  6. "UBICACIÓN"

    Zona en la que encuentra la playa referida

    7. Tipo: String

  7. "LONGITUD-PLAYA"

    Distancia de la playa

    8. Tipo: Integer

  8. "TIPO-PLAYA"

    Indica si la playa es protegida o abierta

    9. Tipo: String

  9. "TIPO-ARENA"

    Indica el tipo de arena (Fina, Gruesa o cantos rodados)

    10. Tipo: String

  10. "LATITUD"

    Determina la latitud de las coordenadas

    11. Tipo: Float

  11. "LONGITUD"

    Determina la longitud de las coordenadas

    12. Tipo: Float

  12. "MAS INFORMACIÓN"

    URL para encontrar más información sobre las playas

    13. Tipo: Url
